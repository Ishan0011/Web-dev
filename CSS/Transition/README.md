# Transition in CSS

CSS transitions are used to create smooth changes between different states of an element.

They control how a CSS property changes over time instead of changing instantly.

Transitions is a process of changing from one to another state.

---

# Transition Shorthand in CSS

The transition shorthand property is used to define multiple transition properties in a single line.

It controls how a CSS property changes smoothly over time.

---

## Transition Shorthand Format

The shorthand includes the following values in order:

- **Property name** – The CSS property to be animated  
- **Duration** – How long the transition takes  
- **Timing function** – The speed curve of the transition  
- **Delay** – Time before the transition starts  
    
- `transition:margin-top 2s ease-in-out 0.2s;`
