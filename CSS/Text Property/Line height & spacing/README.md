# Line Height
The `line-height` property controls the vertical spacing between lines of text.  
Proper line height improves readability, especially in paragraphs.

---

line-height:normal   
line-height:2.5; /2.5*normal   

# Letter Spacing
The `letter-spacing` property adjusts the space between characters in text.  
It is useful for improving text clarity or styling headings.

---

letter-spacing: normal;  
letter-spacing: 10px;
