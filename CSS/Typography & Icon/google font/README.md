# Google Fonts

Google Fonts is a **free library of web-safe fonts** provided by Google.  
It allows developers to easily use **custom fonts** without downloading or hosting font files.

Website: https://fonts.google.com

---

## Why Use Google Fonts?

✔ Free & open-source  
✔ Easy to use  
✔ Optimized for web performance  
✔ Works across all browsers  
✔ Improves typography and UI design  

---

## How to Use Google Fonts (Recommended Method)

### Step 1: Choose a Font
- Visit https://fonts.google.com
- Select a font and required font weights

---

### Step 2: Add Link in HTML

```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600&display=swap" rel="stylesheet">
