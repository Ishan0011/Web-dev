# Typography in CSS

Typography refers to the **style, appearance, and arrangement of text** on a website.  
Good typography improves **readability, accessibility, and visual hierarchy**.

---

## 1. Font Family

Defines the typeface used for text.

```css
font-family: Arial, Helvetica, sans-serif;

# Icons in Web Design

Icons are small visual symbols used to improve UI/UX.  
They help users understand actions quickly (like home, search, delete, menu, etc.).
