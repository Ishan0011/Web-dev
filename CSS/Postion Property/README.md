# Position Property in CSS

The `position:property;` is used to control how an element is positioned on a webpage.

It allows elements to be placed relative to the normal document flow, their parent, or the viewport.

---

## Common Position Values

- **static**  
  Default position. Elements follow the normal document flow.

- **relative**  
  Positioned relative to its normal position.

- **absolute**  
  Positioned relative to the nearest positioned ancestor.

- **fixed**  
  Positioned relative to the viewport and remains fixed during scrolling.

- **sticky**  
  Toggles between relative and fixed based on scroll position.

---

The position property is essential for creating advanced layouts and UI designs.
