# Grid Gap in CSS

Grid gap is used to create space between rows and columns in a CSS Grid layout.

It controls the spacing between grid tracks without affecting the size of grid items.

---

## gap Property

The `gap` property is a shorthand used to define spacing between rows and columns.

- `grid-gap: 10px;`One value sets both row and column gaps.
- `grid-gap: 10px 5px;`Two values set row gap first and column gap second.

---

## row-gap and column-gap

- **row-gap**  
  Sets the space between rows.

- **column-gap**  
  Sets the space between columns.

---

Grid gap helps create clean and visually balanced grid layouts.
