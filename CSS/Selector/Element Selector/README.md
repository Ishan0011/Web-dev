# Element Selector in CSS

The element selector is used to select HTML elements based on their tag name.

It applies styles to all instances of a specific element on a webpage, such as paragraphs, headings, or divs.

h1 {   
    property: value;   
}   

### For multiple tag having a same property   
  
  h1, h3 {   
   property: value;    
  }
