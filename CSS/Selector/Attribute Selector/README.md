# Attribute Selector in CSS

The attribute selector is used to select HTML elements based on the presence or value of an attribute.

It allows styles to be applied without using class or ID selectors and is useful for targeting elements with specific attributes.

Attribute selectors help create flexible and semantic CSS.
    
input[attr="value"] {      
property: value;     
}  
there are other diffent types: [attr~=value] [attr^=value] etc.
