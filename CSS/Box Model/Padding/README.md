# Padding in CSS

The `padding` property is used to create space between the content of an element and its border.

It increases the inner spacing of an element and affects the overall size of the element.

Padding is an important part of the CSS box model and helps improve layout and readability.

They are of:-   
1. padding-left
2. padding-right
3. padding-top
4. padding-bottom 
   
div{   
    padding-left: 50px;  
}   

---
# Padding Shorthand in CSS

The padding shorthand property is used to define padding for all sides of an element in a single line.

Values are applied in a clockwise direction when more than one value is used.

### 1. All 4 sides
Applies the same padding to all sides.  
**padding: 50px**   
    
### 2. Top & Bottom | Left & Right
The first value applies to top and bottom, the second to left and right.
**padding: 1px 2px;**
    
### 3. Top | Left & Right | Bottom
The first value applies to top, the second to left and right, and the third to bottom.
**padding: 1px 2px 3px;**
    
### 4. Top | Right | Bottom | Left (Clockwise)
Values are applied in clockwise order starting from the top.
**padding: 1px 2px 3px 4px;**
