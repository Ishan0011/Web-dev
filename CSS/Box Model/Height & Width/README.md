# Height in CSS

The `height` property is used to define the vertical size of an element.

It specifies how tall the content area of an element content should be. 

div{  
    height: 100px;   
}   

---
   
# Width in CSS

The `width` property is used to define the horizontal size of an element.

It specifies how wide the content area of an element should be.
    
div{  
    width: 100px;   
}
