# Margin in CSS

The `margin` property is used to create space outside the border of an element.

It controls the distance between elements and does not affect the size of the element itself.

they are of:
1. margin-left
2. margin-right
3. margin-top
4. margin-bottom

---
# Margin Shorthand in CSS

The margin shorthand property is used to define margin for all sides of an element in a single line.

When multiple values are used, margins are applied in a clockwise direction.

## Margin Shorthand Formats

### 1. For all four sides
Applies the same margin to all sides.  
**margin: 50px;**   

### 2. Top & Bottom | Left & Right
The first value applies to top and bottom, and the second value applies to left and right.
**margin: 1px 2px;**

### 3. Top | Left & Right | Bottom
The first value applies to top, the second to left and right, and the third to bottom.
**margin: 1px 2px 3px;**

### 4. Top | Right | Bottom | Left (Clockwise)
Values are applied in clockwise order starting from the top.
**margin: 1px 2px 3px 4px;**
