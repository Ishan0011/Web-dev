# Box Model in CSS

The CSS Box Model defines how elements are structured and spaced on a webpage.

Every HTML element is treated as a rectangular box made up of different layers.

---

## Components of the Box Model

- **Width**  
  Defines the horizontal size of the content area.

- **Height**  
  Defines the vertical size of the content area.

- **Padding**  
  The space between the content and the border.

- **Border**  
  Surrounds the padding and content, defining the boundary of the element.

- **Margin**  
  The outer space around the border that separates elements from each other.

---

Understanding the box model is essential for controlling layout, spacing, and alignment in CSS.
