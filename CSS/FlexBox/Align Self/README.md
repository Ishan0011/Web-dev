# align-self in CSS

The `align-self` property is used to align an individual flex item along the cross axis.

It overrides the `align-items` value set on the flex container for a specific item.

---

## Common align-self Values

- **auto**  
  Uses the value defined by `align-items`.

- **stretch**  
  Stretches the item to fill the container along the cross axis.

- **flex-start**  
  Aligns the item at the start of the cross axis.

- **center**  
  Centers the item along the cross axis.

- **flex-end**  
  Aligns the item at the end of the cross axis.

- **baseline**  
  Aligns the item based on its text baseline.

---

align-self provides fine control over individual item alignment in Flexbox.
