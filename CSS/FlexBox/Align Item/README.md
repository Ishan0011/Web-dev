# align-items in CSS

The `align-items` property is used to align flex items along the cross axis of a flex container.

It controls how items are positioned across Cross-axis depending on the flex direction.

---

## Common align-items Values

- **stretch**  
  Items stretch to fill the container along the cross axis.

- **flex-start**  
  Items are aligned at the start of the cross axis.

- **center**  
  Items are centered along the cross axis.

- **flex-end**  
  Items are aligned at the end of the cross axis.

- **baseline**  
  Items are aligned based on their text baseline.

---

align-items is essential for proper alignment in Flexbox layouts.
