# justify-content in CSS

The `justify-content` property is used to align and distribute flex items along the main axis of a flex container.

It controls the spacing between items and their position inside the container.

---

## Common justify-content Values

- **flex-start** `justify-content: flex-start;`  
  Items are aligned at the beginning of the main axis.

- **center** `justify-content: center;`  
  Items are centered along the main axis.

- **flex-end** `justify-content: flex-end;`  
  Items are aligned at the end of the main axis.

- **space-between** `justify-content: space-between;`  
  Items are evenly spaced with no space at the edges.

- **space-around** `justify-content: space-around;`  
  Items are evenly spaced with equal space around them.

- **space-evenly** `justify-content: space-evenly;`  
  Items are evenly spaced with equal space between and around them.

---

justify-content helps create balanced and responsive layouts.
