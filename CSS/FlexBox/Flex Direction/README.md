# flex-direction in CSS

The `flex-direction` property defines the direction in which flex items are placed inside a flex container.

It determines the orientation of the main axis.

---

## Common flex-direction Values

- **row** `flex-direction: row;`    
  Items are placed horizontally from left to right.

- **row-reverse** `flex-direction: row-reverse;`  
  Items are placed horizontally from right to left.

- **column** `flex-direction: column;`  
  Items are placed vertically from top to bottom.

- **column-reverse** `flex-direction: column-reverse;`  
  Items are placed vertically from bottom to top.

---

Changing `flex-direction` also changes the direction of the main axis.
