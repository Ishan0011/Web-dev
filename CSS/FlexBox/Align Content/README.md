# align-content in CSS

The `align-content` property is used to align multiple rows or columns of flex items along the cross axis.

It works only when there is extra space in the container and when flex items wrap onto multiple lines using `flex-wrap`.

---

## Common align-content Values

- **stretch**  
  Lines stretch to fill the available space along the cross axis.

- **flex-start**  
  Lines are packed toward the start of the cross axis.

- **center**  
  Lines are centered along the cross axis.

- **flex-end**  
  Lines are packed toward the end of the cross axis.

- **space-between**  
  Lines are evenly spaced with no space at the edges.

- **space-around**  
  Lines are evenly spaced with space around them.

---

align-content is used to control spacing between rows or columns in wrapped flex layouts.
